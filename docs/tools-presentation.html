시스템 툴 개요

🎓 학생 관리 툴

툴 이름 기능
getStudentsListTool 학원별 학생 명단 조회
displayStudentsListTool 학생 명단 화면 표시
updateStudentListTool 학생 상태 업데이트 (대기/탑승/결석)

🚌 경로 관리 툴

getMultiStopRouteTool

카카오 모빌리티 API 기반 최적 경로 계산

  • 출발지 → 학생 픽업 지점 → 학원 경로 생성
  • 실시간 교통 정보 반영
  • 경로 데이터 JSON 저장

💬 메시징 툴

툴 이름 기능
sendKakaoTalkTool 카카오톡 메시지 전송
sendSlackTool Slack 메시지 전송

주요 알림 내용:

  • 학생 탑승 상태 변경
  • 경로 업데이트
  • 긴급 상황 알림

🎨 시각화 툴

툴 이름 기능
setCanvasTool 캔버스 기본 설정
setMusicCanvasTool 음악 시각화 캔버스 설정
musicCallbackTool 음악 재생 이벤트 처리

🔄 시스템 통합

주요 특징

  • 실시간 데이터 동기화
  • 다중 플랫폼 메시징 통합
  • 직관적인 시각화 인터페이스
  • 최적화된 경로 계산
