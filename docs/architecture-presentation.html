AgenticStoryTeller Architecture

1. 프로젝트 개요

AgenticStoryTeller는 OpenAI의 실시간 API를 활용하여 대화형 스토리텔링과 음악 생성을 제공하는 웹 애플리케이션입니다. 사용자와의 실시간 상호작용을 통해 스토리를 생성하고, 이에 맞는 음악과 시각적 요소를 제공합니다.

2. 기술 스택

React
TypeScript
SCSS
OpenAI Realtime API
WebSocket
Web Audio API

3. 아키텍처 구조

├── components/
│   ├── console/         # 대화 및 이벤트 관련 컴포넌트
│   ├── main/           # 메인 뷰 컴포넌트
│   ├── music/          # 음악 라이브러리 컴포넌트
│   └── visualization/  # 시각화 컴포넌트
├── contexts/
│   └── ConsoleContext  # 전역 상태 관리
├── hooks/
│   ├── useAudioVisualization
│   ├── useRealtimeClient
│   └── useConsoleInitialization
└── tools/
    ├── generateMusic
    ├── generateLyrics
    └── wavtools/

4. 주요 기능

  • 🎯 실시간 대화형 스토리텔링
  • 🎵 AI 기반 음악 생성
  • 🎨 스토리 기반 이미지 생성
  • 📝 실시간 대화 기록
  • 🎼 음악 라이브러리 관리

5. 데이터 흐름

1. 사용자 입력 → ConsoleContext

2. ConsoleContext → OpenAI Realtime API

3. API 응답 → 상태 업데이트

4. 상태 변경 → UI 업데이트

  • 스토리 생성: stories 배열에 저장
  • 음악 생성: music 객체에 저장
  • 이벤트 로깅: realtimeEvents 배열에 저장
